Summary / goal

Replace the current XGBoost P(Redeem) with a GNN that predicts the probability a card-member (CM) redeems an offer within 30 days of an impression. Each row = (cust_xref_id, offer_id, imp_ts, 123 features), label = 1/0. Final product must score impressions and be usable to rank offers per CM (same downstream usage).

High-level approach (why & what)

  • Graph view: Create a bipartite/heterogeneous graph with CMs and Offers as nodes. Impressions are edges (CM Offer) with edge features (timestamp + impression-level features) and the binary label on the edge.

  • Task framing: Edge classification / link classification (predict label on existing impression edges).

  • Why GNN helps: GNNs can learn from local structure (other offers a CM has seen, co-views, similar CMs, offer similarity) and propagate neighborhood signals (spend patterns, cross-offer interactions) that tabular models may not capture easily.

  • Key choices: static vs dynamic/temporal graph, hetero vs simple bipartite, architecture (GraphSAGE/GCN/RelGraphConv/TGN/TemporalGNN), sampling strategy for scale.

Graph construction (practical)

Options ranked by simplicity expressive power:

  1. Simple bipartite static graph (fastest to test)

    • Nodes: CM nodes, Offer nodes.

    • Edges: one edge per impression (CMOffer) with edge features = impression features (123 features or a subset), timestamp, label.

    • Use an edge-predictor that consumes node embeddings + edge features to predict label.

  2. Heterogeneous graph with node types / relation types

    • Same as above, but treat offers and CMs as different node types in a HeteroData object. Use hetero GNN layers (Relational GCN / HeteroGraphConv).

  3. Dynamic/temporal graph (best for time-aware behavior)

    • Keep edges timestamped and use a temporal GNN (TGN, TGAT). Captures evolving behaviour and recency effects.

    • Use for best predictive performance, but more complex to implement and scale.

Practical recommendation to start: Option 1 or 2 (static hetero) to prove uplift; iterate to temporal GNN later.

Feature engineering & inputs

  • Node features

    • CM node: aggregate historical features computed up to impression time (e.g., lifetime spend buckets, recency, frequency, static demographics). If you don't have these, you can aggregate from impressions: avg offer click rate, count impressions in past 30/90 days, etc.

    • Offer node: static offer metadata (category, merchant, discount %, duration, channel).

    • For categorical features use embeddings (learned or precomputed).

  • Edge features

    • Impression-level features (the 123 features). Include imp_ts (or time-since-last-impression) and context (channel, device).

    • If label leakage possible (e.g. future spend features), ensure features are computed only from data available at imp_ts.

  • Normalization & encoding

    • Standardize numeric features.

    • Hash / embed high-cardinality categories (merchant id, offer id) treat offer_id as node identity rather than as edge feature.

Model architecture (practical scaffold)

A strong, incremental plan:

  1. Baseline graph model (edge-classification)

    • Node encoder: separate MLPs for CM and Offer node features node embeddings.

    • GNN layers: GraphSAGE (sampled neighborhood aggregation) or RelGraphConv if you use relation types. Stack 23 layers.

    • Edge predictor: concat(emb_CM, emb_offer, edge_features) MLP sigmoid.

  2. Temporal extension (later)

    • Replace GNN with temporal module (TGN/TGAT) so node embeddings are time-dependent and updated with temporal messages.

  3. Loss

    • Binary cross-entropy with class weighting (because redeems are rare).

    • Optionally focal loss if extreme imbalance.

  4. Ranking objective (optional)

    • Add pairwise ranking loss or listwise loss for direct rank learning (e.g., sampled pairs within same session to optimize NDCG). But start with BCE and evaluate ranking metrics.

Scaling to 2.5M impressions (practical tips)

  • Node count is probably much smaller than impressions (unique CMs and offers). Use node-level batching.

  • Edge-labeled training: use mini-batch edge loaders (NeighborSampler in PyG or EdgeDataLoader in DGL).

  • Sampling methods: GraphSAGE neighbor sampling, Cluster-GCN, or GraphSAINT for large graphs (avoid full-graph training).

  • Feature storage: store node features in memory-mapped files or use on-disk caches. Precompute per-node aggregates if possible.

  • Hardware: multi-GPU if available; if single GPU, use smaller batch sizes and gradient accumulation.

  • Precompute & warm start: you can precompute static node embeddings (e.g., from an MLP on node features) and then only train the edge predictor / fine-tune GNN.

Evaluation: classification + ranking + business

Always evaluate both classification and ranking offline:

  • Classification metrics

    • AUC-ROC, logloss, precision/recall at threshold.

    • GINI (2*AUC - 1), KS.

  • Ranking metrics (critical) you must match how product uses P(Redeem):

    • precision@k, recall@k, NDCG@k, MAP@k compute per card member (session) and average.

    • Evaluate for top-7, top-10, top-20 as you mentioned.

  • Business metrics

    • Expected revenue uplift, incremental click/redeem rate (requires calibration).

  • Statistical tests

    • Bootstrap differences in metrics between XGBoost and GNN to check significance.

  • Offline Online

    • Do an A/B test where you replace ranking with GNN scores vs XGBoost baseline.

Interpretability & feature contribution

  • Use GNNExplainer or Integrated Gradients for GNNs to get local explanations.

  • For model comparison, compute delta in GINI/AUC when removing selected feature groups (NLP features, offer metadata).

  • Use surrogate models: train an XGBoost on GNN embeddings to attribute importance.

# PSEUDO / ADAPTABLE SCAFFOLD (PyTorch + PyG)
import torch
import torch.nn.functional as F
from torch_geometric.nn import SAGEConv, HeteroConv, Linear
from torch_geometric.data import HeteroData
from torch_geometric.loader import NeighborLoader, LinkNeighborLoader  # PyG >=2

class NodeEncoder(torch.nn.Module):
    def __init__(self, in_dim, hid_dim):
        super().__init__()
        self.mlp = torch.nn.Sequential(
            torch.nn.Linear(in_dim, hid_dim),
            torch.nn.ReLU(),
            torch.nn.Linear(hid_dim, hid_dim)
        )
    def forward(self, x): return self.mlp(x)

class GNNModel(torch.nn.Module):
    def __init__(self, cm_dim, offer_dim, hidden_dim=128, num_layers=2, edge_feat_dim=32):
        super().__init__()
        self.cm_encoder = NodeEncoder(cm_dim, hidden_dim)
        self.offer_encoder = NodeEncoder(offer_dim, hidden_dim)
        # Hetero conv example: aggregate messages from offer->cm and cm->offer
        self.conv1 = HeteroConv({
            ('cm', 'views', 'offer'): SAGEConv(hidden_dim, hidden_dim),
            ('offer', 'rev_views', 'cm'): SAGEConv(hidden_dim, hidden_dim),
        }, aggr='mean')
        self.conv2 = HeteroConv({
            ('cm', 'views', 'offer'): SAGEConv(hidden_dim, hidden_dim),
            ('offer', 'rev_views', 'cm'): SAGEConv(hidden_dim, hidden_dim),
        }, aggr='mean')
        # edge feature projector
        self.edge_proj = torch.nn.Sequential(
            torch.nn.Linear(edge_feat_dim, hidden_dim),
            torch.nn.ReLU(),
            torch.nn.Linear(hidden_dim, hidden_dim)
        )
        # final predictor
        self.classifier = torch.nn.Sequential(
            torch.nn.Linear(hidden_dim * 3, hidden_dim),
            torch.nn.ReLU(),
            torch.nn.Linear(hidden_dim, 1)
        )

    def forward(self, x_dict, edge_index_dict, edge_attr):
        # encode nodes
        x_dict = {
            'cm': self.cm_encoder(x_dict['cm']),
            'offer': self.offer_encoder(x_dict['offer'])
        }
        x_dict = self.conv1(x_dict, edge_index_dict)
        x_dict = {k: F.relu(v) for k, v in x_dict.items()}
        x_dict = self.conv2(x_dict, edge_index_dict)
        # edge predictor expects embeddings for the batch edges
        cm_emb = x_dict['cm'][edge_index_dict[('cm', 'views', 'offer')][0]]  # u nodes
        offer_emb = x_dict['offer'][edge_index_dict[('cm', 'views', 'offer')][1]]  # v nodes
        e_emb = self.edge_proj(edge_attr)
        combined = torch.cat([cm_emb, offer_emb, e_emb], dim=1)
        logits = self.classifier(combined).squeeze(-1)
        return logits

End-to-end GNN Pipeline (functional version)

python
import torch
import torch.nn.functional as F
from torch_geometric.data import HeteroData
from torch_geometric.nn import SAGEConv, HeteroConv
from torch_geometric.loader import LinkNeighborLoader
import pandas as pd
import numpy as np
from sklearn.metrics import roc_auc_score
from tqdm import tqdm

# --------------------- 1. LOAD DATA ---------------------
def load_data(dev_path, oot_path):
    dev = pd.read_csv(dev_path)
    oot = pd.read_csv(oot_path)
    print(f"Dev: {len(dev):,} rows | OOT: {len(oot):,} rows")
    return dev, oot

# --------------------- 2. ENCODE IDS ---------------------
def encode_ids(df_train, df_test):
    cm_encoder = {k: i for i, k in enumerate(df_train['cust_xref_id'].unique())}
    offer_encoder = {k: i for i, k in enumerate(df_train['offer_id'].unique())}

    # Map unseen IDs in OOT to new indices
    max_cm = len(cm_encoder)
    max_offer = len(offer_encoder)
    df_train['cm_id'] = df_train['cust_xref_id'].map(cm_encoder)
    df_train['offer_id_enc'] = df_train['offer_id'].map(offer_encoder)
    df_test['cm_id'] = df_test['cust_xref_id'].map(lambda x: cm_encoder.get(x, max_cm + hash(x) % 10000))
    df_test['offer_id_enc'] = df_test['offer_id'].map(lambda x: offer_encoder.get(x, max_offer + hash(x) % 1000))
    return df_train, df_test, cm_encoder, offer_encoder

# --------------------- 3. BUILD GRAPH ---------------------
def build_hetero_graph(df):
    data = HeteroData()

    cm_features = df.groupby('cm_id').mean(numeric_only=True)
    offer_features = df.groupby('offer_id_enc').mean(numeric_only=True)

    data['cm'].x = torch.tensor(cm_features.values, dtype=torch.float)
    data['offer'].x = torch.tensor(offer_features.values, dtype=torch.float)

    edge_index = torch.tensor(df[['cm_id', 'offer_id_enc']].values.T, dtype=torch.long)
    edge_attr = torch.tensor(df.drop(columns=['cust_xref_id', 'offer_id', 'imp_ts', 'label', 'cm_id', 'offer_id_enc']).values, dtype=torch.float)
    edge_label = torch.tensor(df['label'].values, dtype=torch.float)

    data['cm', 'views', 'offer'].edge_index = edge_index
    data['cm', 'views', 'offer'].edge_attr = edge_attr
    data['cm', 'views', 'offer'].edge_label = edge_label
    return data

# --------------------- 4. DEFINE MODEL FUNCTIONS ---------------------
def build_layers(in_dim, hidden_dim):
    return torch.nn.ModuleList([
        SAGEConv(in_dim, hidden_dim),
        SAGEConv(hidden_dim, hidden_dim)
    ])

def forward_pass(x_dict, edge_index_dict, edge_attr, layers, edge_mlp):
    x_cm, x_offer = x_dict['cm'], x_dict['offer']
    for layer in layers:
        x_cm = F.relu(layer(x_cm, edge_index_dict[('cm', 'views', 'offer')]))
        x_offer = F.relu(layer(x_offer, edge_index_dict[('cm', 'views', 'offer')].flip(0)))
    x_dict = {'cm': x_cm, 'offer': x_offer}

    src, dst = edge_index_dict[('cm', 'views', 'offer')]
    cm_emb = x_dict['cm'][src]
    offer_emb = x_dict['offer'][dst]
    e_emb = torch.cat([cm_emb, offer_emb, edge_attr], dim=1)
    return edge_mlp(e_emb).squeeze()

def create_model(in_dim, hidden_dim):
    layers = build_layers(in_dim, hidden_dim)
    edge_mlp = torch.nn.Sequential(
        torch.nn.Linear(2 * hidden_dim + in_dim, hidden_dim),
        torch.nn.ReLU(),
        torch.nn.Linear(hidden_dim, 1)
    )
    return layers, edge_mlp

# --------------------- 5. TRAIN ---------------------
def train_model(data, layers, edge_mlp, epochs=3, lr=1e-3, batch_size=2048):
    device = torch.device('cuda' if torch.cuda.is_available() else 'cpu')
    layers, edge_mlp = layers.to(device), edge_mlp.to(device)
    data = data.to(device)

    opt = torch.optim.Adam(list(layers.parameters()) + list(edge_mlp.parameters()), lr=lr)
    loader = LinkNeighborLoader(
        data,
        edge_label_index=(('cm', 'views', 'offer'), data['cm', 'views', 'offer'].edge_index),
        edge_label=data['cm', 'views', 'offer'].edge_label,
        batch_size=batch_size,
        shuffle=True
    )

    for epoch in range(epochs):
        total_loss = 0
        for batch in tqdm(loader, desc=f"Epoch {epoch+1}/{epochs}"):
            batch = batch.to(device)
            logits = forward_pass(batch.x_dict, batch.edge_index_dict,
                                  batch['cm', 'views', 'offer'].edge_attr,
                                  layers, edge_mlp)
            labels = batch['cm', 'views', 'offer'].edge_label
            loss = F.binary_cross_entropy_with_logits(logits, labels)
            opt.zero_grad()
            loss.backward()
            opt.step()
            total_loss += loss.item()
        print(f"Epoch {epoch+1} | Loss = {total_loss/len(loader):.4f}")

    return layers, edge_mlp

# --------------------- 6. SCORING ---------------------
def score_edges(df, cm_encoder, offer_encoder, layers, edge_mlp):
    device = torch.device('cuda' if torch.cuda.is_available() else 'cpu')
    df['cm_id'] = df['cust_xref_id'].map(lambda x: cm_encoder.get(x, -1))
    df['offer_id_enc'] = df['offer_id'].map(lambda x: offer_encoder.get(x, -1))
    df = df[(df['cm_id'] != -1) & (df['offer_id_enc'] != -1)]

    with torch.no_grad():
        x_cm = torch.randn(len(cm_encoder), 64, device=device)
        x_offer = torch.randn(len(offer_encoder), 64, device=device)
        x_dict = {'cm': x_cm, 'offer': x_offer}
        edge_index_dict = {('cm', 'views', 'offer'): torch.tensor(df[['cm_id','offer_id_enc']].values.T, dtype=torch.long, device=device)}
        edge_attr = torch.tensor(df.drop(columns=['cust_xref_id','offer_id','imp_ts','label','cm_id','offer_id_enc']).values, dtype=torch.float, device=device)
        logits = forward_pass(x_dict, edge_index_dict, edge_attr, layers, edge_mlp)
        probs = torch.sigmoid(logits).cpu().numpy()
    df['pred'] = probs
    return df

# --------------------- 7. EVALUATION ---------------------
def evaluate(df):
    y_true, y_pred = df['label'], df['pred']
    auc = roc_auc_score(y_true, y_pred)
    gini = 2 * auc - 1

    def precision_at_k(k):
        return (df.sort_values('pred', ascending=False).head(k)['label'].mean())

    print(f"AUC: {auc:.4f} | GINI: {gini:.4f}")
    print("Precision@7:", precision_at_k(7))
    print("Precision@10:", precision_at_k(10))
    print("Precision@20:", precision_at_k(20))

# --------------------- 8. MAIN ---------------------
def main():
    dev, oot = load_data("dev.csv", "oot.csv")
    dev, oot, cm_enc, offer_enc = encode_ids(dev, oot)
    data = build_hetero_graph(dev)

    in_dim = data['cm', 'views', 'offer'].edge_attr.shape[1]
    hidden_dim = 64

    layers, edge_mlp = create_model(in_dim, hidden_dim)
    layers, edge_mlp = train_model(data, layers, edge_mlp, epochs=5)

    oot_scored = score_edges(oot, cm_enc, offer_enc, layers, edge_mlp)
    evaluate(oot_scored)

if __name__ == "__main__":
    main()

    data = HeteroData()

    # Average numeric features per cm and per offer to form simple node features
    cm_features = df.groupby('cm_id').mean(numeric_only=True)
    offer_features = df.groupby('offer_id_enc').mean(numeric_only=True)

    # Assign node features (these can later be replaced by better features)

    # The label (1 = redeemed, 0 = not redeemed)
    edge_label = torch.tensor(df['label'].values, dtype=torch.float)

    # Add this information to the graph
    data['cm', 'views', 'offer'].edge_index = edge_index
    data['cm', 'views', 'offer'].edge_attr = edge_attr
    data['cm', 'views', 'offer'].edge_label = edge_label

    return data

# ============================================================
def build_layers(in_dim, hidden_dim):
    # Build two GraphSAGE layers that will learn from neighboring nodes
    return torch.nn.ModuleList([
        SAGEConv(in_dim, hidden_dim),
        SAGEConv(hidden_dim, hidden_dim)
    ])

def forward_pass(x_dict, edge_index_dict, edge_attr, layers, edge_mlp):
    # Extract CM and offer node feature matrices
    x_cm, x_offer = x_dict['cm'], x_dict['offer']

    # Pass node features through two GraphSAGE layers
    for layer in layers:
        x_cm = F.relu(layer(x_cm, edge_index_dict[('cm', 'views', 'offer')]))           # CM -> Offer
        x_offer = F.relu(layer(x_offer, edge_index_dict[('cm', 'views', 'offer')].flip(0)))  # Offer -> CM

    # Combine them back into a dictionary
    x_dict = {'cm': x_cm, 'offer': x_offer}

    # For each edge (impression), get embeddings of its CM and Offer nodes
    src, dst = edge_index_dict[('cm', 'views', 'offer')]
    cm_emb = x_dict['cm'][src]
    offer_emb = x_dict['offer'][dst]

    # Concatenate both embeddings + the edge features
    e_emb = torch.cat([cm_emb, offer_emb, edge_attr], dim=1)

    # Pass this through an MLP to predict a single score (logit)
    return edge_mlp(e_emb).squeeze()

def create_model(in_dim, hidden_dim):
    # Create two GNN layers + a simple feedforward MLP for prediction
    layers = build_layers(in_dim, hidden_dim)
    edge_mlp = torch.nn.Sequential(
        torch.nn.Linear(2 * hidden_dim + in_dim, hidden_dim),
        torch.nn.ReLU(),
        torch.nn.Linear(hidden_dim, 1)
    )
    return layers, edge_mlp


# ============================================================
# 5. TRAINING FUNCTION
# ============================================================
def train_model(data, layers, edge_mlp, epochs=3, lr=1e-3, batch_size=2048):
    device = torch.device('cuda' if torch.cuda.is_available() else 'cpu')
    layers, edge_mlp = layers.to(device), edge_mlp.to(device)
    data = data.to(device)

    # Use Adam optimizer
    opt = torch.optim.Adam(list(layers.parameters()) + list(edge_mlp.parameters()), lr=lr)

    # DataLoader that samples edges in mini-batches (since we have millions of edges)
    loader = LinkNeighborLoader(
        data,
        edge_label_index=(('cm', 'views', 'offer'), data['cm', 'views', 'offer'].edge_index),
        edge_label=data['cm', 'views', 'offer'].edge_label,
        batch_size=batch_size,
        shuffle=True
    )

    # Training loop
    for epoch in range(epochs):
        total_loss = 0
        for batch in tqdm(loader, desc=f"Epoch {epoch+1}/{epochs}"):
            batch = batch.to(device)

            # Forward pass
            logits = forward_pass(
                batch.x_dict, batch.edge_index_dict,
                batch['cm', 'views', 'offer'].edge_attr,
                layers, edge_mlp
            )

            # Compute loss using binary cross entropy
            labels = batch['cm', 'views', 'offer'].edge_label
            loss = F.binary_cross_entropy_with_logits(logits, labels)

            # Backpropagation
            opt.zero_grad()
            loss.backward()
            opt.step()

            total_loss += loss.item()

        print(f"Epoch {epoch+1} | Average Loss: {total_loss/len(loader):.4f}")

    return layers, edge_mlp


# ============================================================
# 6. SCORING FUNCTION (GET PREDICTION PROBABILITIES)
# ============================================================
def score_edges(df, cm_encoder, offer_encoder, layers, edge_mlp):
    device = torch.device('cuda' if torch.cuda.is_available() else 'cpu')

    # Convert CM and Offer IDs to their numeric IDs (drop unknowns)
    df['cm_id'] = df['cust_xref_id'].map(lambda x: cm_encoder.get(x, -1))
    df['offer_id_enc'] = df['offer_id'].map(lambda x: offer_encoder.get(x, -1))
    df = df[(df['cm_id'] != -1) & (df['offer_id_enc'] != -1)]

    # No full graph during inference  use precomputed embeddings or dummy ones
    with torch.no_grad():
        x_cm = torch.randn(len(cm_encoder), 64, device=device)
        x_offer = torch.randn(len(offer_encoder), 64, device=device)
        x_dict = {'cm': x_cm, 'offer': x_offer}
        edge_index_dict = {
            ('cm', 'views', 'offer'): torch.tensor(
                df[['cm_id', 'offer_id_enc']].values.T,
                dtype=torch.long, device=device)
        }

        edge_attr = torch.tensor(
            df.drop(columns=['cust_xref_id', 'offer_id', 'imp_ts', 'label', 'cm_id', 'offer_id_enc']).values,
            dtype=torch.float, device=device
        )

        # Get logits and convert to probabilities
        logits = forward_pass(x_dict, edge_index_dict, edge_attr, layers, edge_mlp)
        probs = torch.sigmoid(logits).cpu().numpy()

    df['pred'] = probs
    return df


# ============================================================
# 7. EVALUATION FUNCTION
# ============================================================
def evaluate(df):
    y_true, y_pred = df['label'], df['pred']

    # Classification metrics
    auc = roc_auc_score(y_true, y_pred)
    gini = 2 * auc - 1

    # Quick top-K precision check (overall, not per CM)
    def precision_at_k(k):
        return (df.sort_values('pred', ascending=False).head(k)['label'].mean())

    print(f"AUC: {auc:.4f} | GINI: {gini:.4f}")
    print("Precision@7:", precision_at_k(7))
    print("Precision@10:", precision_at_k(10))
    print("Precision@20:", precision_at_k(20))
